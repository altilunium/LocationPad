# Altilunium LocationPad

<img width="1365" height="659" alt="image" src="https://github.com/user-attachments/assets/99fc804b-7934-4528-bf9d-bcada9a62a20" />


A lightweight, offline-friendly web app for marking, labeling, and revisiting meaningful places on an OpenStreetMap-based map. Designed for quick place logging, personal mapping, and spatial note-taking without accounts or servers.
